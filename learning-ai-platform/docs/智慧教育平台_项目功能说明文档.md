# 智慧教育平台 - 项目功能说明文档

## 一、项目概述
“智慧教育平台”是一个基于前后端分离架构（Vue3 + Node.js/Express + MongoDB）的现代化在线学习与评测系统。该平台致力于为管理员、教师、学生提供一体化的教育服务，支持课程学习、AI辅助教学、在线评测、数据统计等核心功能。

## 二、核心功能清单

### 1. 用户与权限管理
- **角色体系**：支持管理员、教师、学生等多角色登录。
- **权限控制**：基于JWT的身份认证机制，不同角色展示不同的功能菜单。
- **账号管理**：支持用户注册、登录、个人信息修改、密码重置等。

### 2. 课程与学习模块
- **课程展示与学习**：学生可浏览课程列表、观看教学视频、查阅图文资料。
- **AI辅助学习**：集成AI智能助手，为学生提供代码分析、学习路线生成、智能问答等功能。
- **学习进度追踪**：自动记录学生的学习进度（时长、完成的课时等）。

### 3. 在线评测与作业系统
- **题库管理**：支持单选题、多选题、编程题等多种题型。
- **在线考试/练习**：学生可在线答题，提交后由系统自动进行判分。
- **错题本与能力图谱**：根据学生的答题情况，自动收集错题并生成能力雷达图，帮助学生查漏补缺。

### 4. 互动与社区模块
- **讨论区/动态推文**：学生与教师可以发布学习动态（推文）、相互点赞和评论，形成良好的学习氛围。
- **群组功能**：支持创建学习小组，方便学生之间进行讨论与资料分享。

### 5. 积分与钱包系统
- **虚拟钱包**：用户在平台中的互动和学习可以获取积分（或金币），用于兑换增值服务。
- **会员体系**：支持普通用户与VIP用户，VIP用户享有更多高级课程及AI调用次数。

## 三、技术实现亮点
- **前端架构**：采用 Vue3 + Pinia + Vue Router + Element Plus，保证了优秀的页面响应速度和UI一致性。
- **后端架构**：采用 Node.js + Express，配合 MongoDB 进行数据存储，并使用 Redis 进行高频数据的缓存。
- **安全机制**：
  - 密码使用 bcryptjs 进行加密存储。
  - 使用 express-rate-limit 防御暴力破解。
  - 数据输入经过严格的 xss 清理和格式校验。
- **离线与缓存**：前端实现了部分数据（如安全 LocalStorage）的本地缓存策略，保障在弱网环境下的可用性。

## 四、部署与运行
- 项目已包含完整的 `start-services.ps1` 和 `ecosystem.config.json` 等脚本，支持快速本地启动和 PM2 生产环境部署。
- 后端接口规范统一遵循 RESTful 风格，返回标准 JSON 结构：`{ "code": 200, "msg": "成功", "data": {} }`。
