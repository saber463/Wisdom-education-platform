# 算法可视化组件优化文档

## 1. 优化背景
为了提升算法可视化实验室的用户体验，我们在原有条形图的基础上进行了布局优化和数值显示增强。主要解决了在不同数据量下数值显示不清晰、对齐不准确以及在大数据量下布局拥挤的问题。

## 2. 主要修改内容

### 2.1 数值显示增强
- **动态数值标签**：
  - 当数据量（`arraySize`）小于等于 40 时，在每个柱状图下方直接显示具体的数值。
  - 当数据量大于 40 时，采用抽样显示策略（每隔 5 个元素显示一个数值和标记点），避免标签重叠。
- **交互增强**：
  - 在数据量较小（<= 25）时，增加了鼠标悬停（Hover）在柱子上时顶部浮动显示数值的功能。
  - 数值颜色根据状态动态变化：激活状态（蓝色）、已排序状态（绿色）、默认状态（灰色）。

### 2.2 布局与响应式优化
- **响应式宽度与间距**：
  - 引入了 `barWidth` 和 `barMargin` 的计算逻辑，根据数据量自动调整柱子的宽度和外边距。
  - | 数据量范围 | 柱子宽度 | 间距 |
    | :--- | :--- | :--- |
    | <= 20 | 30px | 6px |
    | 21 - 40 | 20px | 3px |
    | 41 - 60 | 12px | 2px |
    | > 60 | 6px | 1px |
- **容器优化**：
  - 增加了 `overflow-x-auto` 和自定义滚动条样式，确保在极大数据量下依然可以横向滑动查看。
  - 使用 `flex items-end justify-center` 确保所有柱子在底部对齐且整体居中。

### 2.3 视觉细节提升
- **动画平滑度**：将过渡时间从 `200ms` 调整为 `300ms`，并使用 `cubic-bezier` 曲线，使排序过程更丝滑。
- **高亮效果**：为正在比较的柱子增加了脉冲动画（Pulse）和增强的阴影效果，提升视觉辨识度。
- **UI 卡片**：增强了 `glass-card` 的模糊度和边框透明度，提升了整体的极客感。

## 3. 技术实现细节
- **文件路径**：`src/views/learning/AlgoViz.vue`
- **核心组件**：Vue 3 Composition API (`script setup`)
- **样式框架**：Tailwind CSS + 自定义 CSS

## 4. 验证建议
1. 切换不同的算法演示，观察柱子颜色和数值的变化。
2. 调整“数据量”滑块，观察柱子宽度、间距以及底部数值标签的自动适配情况。
## 5. 新增算法支持 (2026-04-17)
为了进一步丰富实验室功能，我们扩展了以下算法的可视化：
- **计数排序 (Counting Sort)**：非比较类排序，展示桶计数的原理。
- **基数排序 (Radix Sort)**：基于位的排序，展示分拣和收集的过程。
- **线性搜索 (Linear Search)**：基础搜索算法，展示逐个排查的过程。
- **目标值交互**：搜索算法（二分、线性）现在共用统一的目标值随机生成逻辑，并在 UI 中高亮显示。
