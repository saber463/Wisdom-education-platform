# 智慧教育平台 - 项目介绍与演示文档

## 1. 项目详情及使用方式
**项目名称**：智慧教育平台 (Wisdom Education Platform)
**项目定位**：一个集 AI 辅助教学、个性化推荐、在线评测与社区互动于一体的现代化 Web 学习平台。

### 使用方式：
1. **访问地址**：项目已部署于云端服务器，用户可通过浏览器直接访问。
2. **注册/登录**：用户可选择“学生”、“教师”或“家长”角色进行注册。
3. **个性化配置**：首次登录后，系统会弹出兴趣选择窗口（包含机器学习、大语言模型、前端开发等），用户选择后，平台将自动调整推荐内容。
4. **核心流程**：
   - **学生**：在首页查看推荐推文和课程 -> 进入知识库学习 -> 使用 AI 助手进行代码分析或问答 -> 完成在线测评 -> 查看能力图谱。
   - **教师**：发布课程资料 -> 管理学生进度 -> 在社区发布技术动态。
   - **家长**：监控孩子的学习时长与能力成长曲线。

## 2. 项目亮点
- **AI 深度融合**：内置 AI 智能助手，支持代码实时诊断、个性化学习路径生成及智能问答，为每位学生提供 7x24 小时的私人导师。
- **“千人千面”推荐算法**：基于用户自选兴趣标签，利用后端过滤与前端加权排序算法，实现学习社区（推文）与知识库内容的个性化精准推荐。
- **全栈自动化部署与合规性**：代码托管于 GitHub，通过 Nginx 反向代理与 PM2 进程守护，在云服务器上实现高可用运行。**项目域名已完成 ICP 备案，确保了在国内互联网环境下的合规访问与稳定性。**
- **可视化能力图谱**：根据测评数据自动生成雷达图，直观展示用户的知识盲区与长处。
- **商业闭环设计**：集成虚拟钱包与 VIP 会员体系，通过积分/金币购买增值 AI 服务，具备清晰的商业化潜力。

## 3. Web 网页使用情况概述
- **响应式设计**：完美适配 PC 端与移动端浏览器，采用 Element Plus 实现现代化的 UI/UX 交互。
- **极速响应**：前端利用 Vite 构建，结合 Redis 后端缓存，首页及核心接口加载速度达到毫秒级。
- **高活跃社区**：学习社区支持推文发布、Logo 自动匹配、点赞评论，形成了活跃的知识分享氛围。

## 4. 项目涉及的关键技术
- **前端**：Vue 3 (Composition API), Pinia (状态管理), Vue Router, Element Plus, Vite, ECharts (图表展示)。
- **后端**：Node.js, Express, JWT (身份认证), RESTful API 规范。
- **数据库**：MongoDB (非关系型数据库), Redis (高性能缓存)。
- **AI 接入**：对接通义千问/大语言模型 API，Prompt 提示词工程。
- **工程化/部署**：GitHub (版本控制), Nginx (Web 服务器/反向代理), PM2 (进程管理), 云服务器 (Linux)。

## 7. 优化说明
本项目是在基础教学平台框架上进行的深度二次开发与优化，主要优化点包括：
1. **推荐算法升级**：从单一的列表展示优化为基于用户 `learningInterests` 标签的动态推荐算法（覆盖社区与知识库）。
2. **AI 功能落地**：新增了代码分析、智能路径规划等多个 AI 核心模块。
3. **数据架构重构**：引入 Redis 缓存层，并优化了 MongoDB 索引，提升了千万级数据检索效率。
4. **UI/UX 体验翻新**：重构了推文展示逻辑（自动匹配官方 Logo），并修复了头像上传、硬编码名称等多个用户体验痛点。
5. **部署自动化与合规化**：实现了从本地开发到 GitHub 云端部署的全流程自动化，**并完成了域名的 ICP 备案，满足了商业化上线的政策要求。**
