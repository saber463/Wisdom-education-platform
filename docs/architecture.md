## 整体架构

```
┌─────────────────────────────────────────────────────────┐

│              ┌───────▼────────┐                          │

│              └────────────────┘                          │
└─────────────────────────────────────────────────────────┘
                       │
                       │ HTTP/WebSocket
                       │
┌─────────────────────▼─────────────────────────────────┐
│              Node.js后端服务 (Express)                 │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐            │
│  │ 认证模块 │  │ 作业模块 │  │ 批改模块 │            │
│  └──────────┘  └──────────┘  └──────────┘            │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐            │
│  │ 学情分析 │  │ 推荐模块 │  │ 通知模块 │            │

         │                              │
         │ gRPC                         │ gRPC
         │                              │
```

## 技术栈

- **前端**: Vue 3 + Vite + TypeScript + Rust-WASM
- **后端**: Node.js 18+ + Express + gRPC
- **AI服务**: Python 3.10+ + Flask + BERT
- **高性能服务**: Rust 1.70+ + Actix-web + gRPC
- **数据库**: MySQL 8.0

## 服务通信

- 前端 ↔ Node后端: HTTP/WebSocket
- 前端 ↔ WASM: 直接调用
- Node ↔ Python: gRPC (端口50051)
- Node ↔ Rust: gRPC (端口50052)

## 部署环境

纯Windows 10/11本地环境，禁用Docker容器化。
