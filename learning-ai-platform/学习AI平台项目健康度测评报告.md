## 一、项目概述

学习AI平台是一个基于 Vue.js 前端和 Express.js 后端的全栈学习管理平台，采用前后端分离架构。平台提供用户认证、课程学习、测试评估、学习路径生成、社区互动、会员系统等核心功能模块，旨在为用户提供个性化的AI驱动学习体验。

### 1.1 技术架构

**前端技术栈**:
- Vue 3.5.26 (响应式框架)
- Vite 7.3.0 (构建工具)
- Pinia 3.0.4 (状态管理)
- Vue Router 4.6.4 (路由管理)
- Element Plus 2.13.0 (UI组件库)
- Tailwind CSS 3.4.19 (CSS框架)
- Axios 1.13.2 (HTTP客户端)
- PWA支持 (离线访问)

**后端技术栈**:
- Express 4.22.1 (Web框架)
- MongoDB + Mongoose 8.20.1 (数据库)
- Redis 5.10.0 (缓存系统)
- JWT (jsonwebtoken 9.0.2) (认证机制)
- bcryptjs 2.4.3 (密码加密)
- Sharp 0.33.0 (图像处理)
- Multer 2.0.2 (文件上传)

**开发工具**:
- ESLint 9.39.2 (代码规范)
- Prettier 3.7.4 (代码格式化)
- TypeScript 5.9.3 (类型检查)
- Jest 29.7.0 (测试框架)
- Nodemon 3.1.4 (开发服务器)
